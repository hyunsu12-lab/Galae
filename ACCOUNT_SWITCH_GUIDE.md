# 계정 전환 가이드

## 방법 1: 앱 내에서 로그아웃 (권장)

1. 페이지 상단 네비게이션 바에서 **"로그아웃"** 버튼 클릭
2. 자동으로 로그인 페이지로 이동
3. 다른 계정의 이메일과 비밀번호로 로그인

## 방법 2: 브라우저 개발자 도구 사용

### Chrome/Edge
1. F12 또는 Cmd+Option+I (Mac) / Ctrl+Shift+I (Windows)로 개발자 도구 열기
2. **Application** 탭 클릭
3. 왼쪽 메뉴에서 **Local Storage** 또는 **Session Storage** 확장
4. `http://localhost:5173` (또는 현재 도메인) 클릭
5. Firebase 관련 키 삭제:
   - `firebase:authUser:...` 키 삭제
6. 페이지 새로고침 (F5)

### Firefox
1. F12로 개발자 도구 열기
2. **Storage** 탭 클릭
3. **Local Storage** 또는 **Session Storage** 확장
4. Firebase 관련 키 삭제
5. 페이지 새로고침

## 방법 3: 브라우저 쿠키/저장소 삭제

### Chrome/Edge
1. 주소창 왼쪽의 자물쇠 아이콘 클릭
2. **쿠키 및 사이트 데이터** 클릭
3. **모두 삭제** 클릭
4. 페이지 새로고침

### Firefox
1. 주소창 왼쪽의 자물쇠 아이콘 클릭
2. **연결 정보** 클릭
3. **쿠키 삭제** 클릭
4. 페이지 새로고침

## 방법 4: 시크릿/프라이빗 모드 사용

1. 새 시크릿/프라이빗 창 열기
2. 앱 URL 접속
3. 다른 계정으로 로그인

## 현재 로그인된 계정 확인

페이지 상단 네비게이션 바에서 현재 로그인된 이메일 주소를 확인할 수 있습니다.

## 문제 해결

### 로그아웃 후에도 이전 계정이 보이는 경우

1. 브라우저 캐시 삭제
2. 하드 새로고침: 
   - Windows: Ctrl+Shift+R
   - Mac: Cmd+Shift+R
3. 브라우저 완전 종료 후 다시 시작

### 다른 계정으로 로그인할 수 없는 경우

1. Firebase Console에서 해당 이메일이 등록되어 있는지 확인
2. 비밀번호가 올바른지 확인
3. 계정이 비활성화되지 않았는지 확인
